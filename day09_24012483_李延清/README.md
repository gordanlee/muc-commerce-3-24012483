## 用自己的话回答

- 什么是特征，什么是标签：特征是模型可以查看的用户信息（如使用月数、登录方式），标签是希望预测的答案（流失/未流失）。
- 为什么要保留测试集：测试集模拟未经见过的用户，只有在测试集上评估才能知道模型对新数据是否有效，避免过拟合。
- 为什么83%准确率仍可能没有用：因为最低参照线永远预测"未流失"，流失召回率为0，找不到任何流失用户，对主动干预毫无价值。
